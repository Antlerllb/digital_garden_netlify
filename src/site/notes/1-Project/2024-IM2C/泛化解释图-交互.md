# 1 内容
画点图，把人的交互（例如抚摸、喂食等）体现出来。
# 2 其他例子
比如我要把网球泛化到篮球
左图（绘图）：我就把篮球画个图，加点箭头，再解释球之间的动力学是差不多的。
右图（代码）：x轴为抚摸，y轴为喂食，z轴为幸福感（体现领养意愿），正相关关系。
