# 1 文件处理与信息统计
- process（得到text_set.csv）
	- 对于每个数据集，在output内建一个新文件夹
	- 将各类型的文件整理成csv
	- 必要字段：'author_id', 'created_date', 'text'
	- 其他字段：都保留，便于后续增删
- statistics
	- 统计各类别的平均单词数量：类别id、类别、平均单词数量
	- 统计各作者的version：作者id、version数量（可以是不同的作业主题）
