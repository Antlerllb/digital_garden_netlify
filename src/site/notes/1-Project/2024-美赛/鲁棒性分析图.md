# 3 Prompt
我们的模型对于不同的外部参数具有较强的稳健性。例如球场材质，Hard Court和Grass Court。
同时，我们的模型对数据的规模需求较高，当数据越多时，模型效果会得到更多的gain。
当φφ足够小时。参数φφ对预测过程的保真度和控制时间均有显著影响。通过解析计算，我们揭示了控制时间在φφ趋于零时呈现对数发散(参见补充章节II)。
