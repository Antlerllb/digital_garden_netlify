# 1 摘要
风传入侵植物因其快速的扩散对本地生物多样性造成显著威胁。目前对于风传植物的种群扩散多使用MaxEnt等模型在宏观尺度研究。本研究分析了典型风传植物——千里光，并评价其在四川地区的入侵性，基于逻辑斯谛方程和高斯烟雨模型建立了微观尺度的扩散动态模型和种群增长函数，为类似风传入侵植物防治提供了预测方法和防治指导。
